Design decisions
  • Not pure black: wine surfaces keep reds alive and reduce harsh contrast.
  • Crimson is sparse: only focus/active/danger — not always-on neon.
  • Audit readability: thin borders + subtle stripes, no heavy glass.
  • Focus-visible: every interactive element gets a visible ring.
  • Motion restraint: no heavy animation; respects reduced motion.
