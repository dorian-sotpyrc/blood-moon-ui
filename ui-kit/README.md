# Blood Moon UI — Minimal PLEX Kit

Open `index.html` (no build steps, no dependencies).

## What’s included
- Navbar
- Cards / panels / containers
- Inputs (focus, error, disabled)
- Buttons (primary/secondary/ghost/danger/icon/disabled)
- Badges + chips
- Styled list + audit rows
- Optional table (scrollable)

## Token-driven
Most styling is controlled via `css/tokens.css`.

- `assets/img/hero.webp` — Blood Moon hero
- `assets/img/preview-1.webp`, `preview-2.webp` — preview tiles
